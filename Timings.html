Conclusions

Jarvis march performs better on an average, however, the difference between them is not significant, in the sense that they both run in the same order of time. However, Kirkpartik-Sidel does not seem to scale up to that factor due to the fact that the use of vectors is not efficient due to the large number of memory operations.
